# BeamDrop Design

This directory contains BeamDrop brand, UI system, screen, flow, and native app
experience documentation.

Design work must reflect native platform conventions and the privacy-first,
local-first transfer model. BeamDrop should feel fast, premium, minimal, native,
private, and professional.
